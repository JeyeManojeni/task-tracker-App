# Task Tracker

A full-stack task management application built with Node.js, Express, SQLite, and Vanilla JavaScript.

## Setup and Run Instructions

1. **Install Dependencies:**
   - Backend: Navigate to `backend/` and run `npm install`.
   - Frontend: No installation needed (Vanilla JS).

2. **Start the Backend:**
   - In `backend/`, run `npm start`. The server will run on `http://localhost:3000`.

3. **Start the Frontend:**
   - Open `frontend/index.html` in a web browser (e.g., via a local server like `python -m http.server` or directly if CORS allows).

4. **Usage:**
   - Create tasks via the form.
   - Filter and sort tasks using dropdowns.
   - Update task status/priority inline.
   - View insights in the panel.
